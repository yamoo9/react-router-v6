# 라우트 구성(Route Config)

`useRoutes` 훅을 사용해 라우트 구성 객체를 사용해 라우트를 구성하는 방법을 학습합니다.

- [ ] `useRoutes` 훅을 사용해 라우트 구성

| URL 경로       | 컴포넌트  |
| -------------- | --------- |
| /              | Home      |
| /books         | Books     |
| /books/:isbn   | Book      |
| /consumers     | Consumers |
| /consumers/:id | Consumer  |
